# 🚀React Router Practice Project

A beginner-friendly React project built to learn and practice **React Router DOM** concepts. This project demonstrates how routing works in a Single Page Application (SPA) using modern React.

---

## 📌 Features

- 🏠 Home Page
- 👨‍💻 About Page
- 📞 Contact Page
- 🌐 GitHub Page
- 🔗 Navigation using `Link`
- 🎯 Active Navigation using `NavLink`
- 📂 Nested Routing with `Outlet`
- ⚡ Dynamic Routing using `useParams`
- 📡 Route Loaders with `loader`
- 🚀 Single Page Application (SPA) Navigation
- 📱 Responsive UI

---

## 🚀 Installation

Clone the repository

```bash
git clone <daimx1214@gmail.com>
```

Move into the project folder

```bash
cd 07reactRouter
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
